## 天赋云巢

- **游戏人数：**  2~8
- **原作：**  oh collei

### 游戏简介
- 本游戏基于云顶之巢改编，游戏的目标是通过在盘面中摆放卡牌构建连线，并在对战中击败其他玩家，成为最后存活者。
- 玩家需要在有限棋盘中规划布局，同时利用卡牌、天赋与随机事件形成优势。

### 游戏流程
- 基础连线得分采用数字蜂巢规则：玩家每回合需要把一块数字砖块填入某个方格中（砖块允许被覆盖），当某条竖线或者斜线均为一种数字即可得分，得分数量为该数字乘以直线长度。**但如果一行内仅包含癞子线，得分为长度×10。0数字线可以参与连线并计入连线数，但该连线得分为0。**
- 每名玩家初始血量为 **150**。从第3回合起，正常轮时结束随机挑选一名对手（奇数名玩家时会有一位玩家对战镜像），比拼当前分数，较低者会减去分数之差的血量（镜像的血量不会影响玩家本体）。
- 第8、15、22等轮（回合数+7）会从"卡池2"选出等于(人数+1)枚砖块并且公示（卡池2不包含癞子，若卡池2已空，则重置卡池重新抽取），由血量低往高进行选择，如果血量相同，则分数更低的玩家先进行选择。如果分数相同，由系统随机选择顺序。
- 血量归零玩家淘汰，剩余 1 人时游戏结束；卡池耗尽时所有存活者并列获胜。
- **最终积分完全按淘汰名次结算（和总分无关）**：每比一名玩家晚淘汰 +100 分，每比一名玩家早淘汰 −100 分。
    + 例如：4 人局完整分胜负的常规结果为 −300 / −100 / +100 / +300。
    + 同回合同阶段（同为对战、或同为中毒）淘汰的玩家名次完全相同、分数相同。
    + 中毒结算晚于对战，因此中毒淘汰名次 **优于** 同回合的对战阶段淘汰。
    + 卡池耗尽时所有存活玩家并列第一。

### 天赋系统

- 每完成 **3条连线** 可获得一次天赋选择，**破坏后重连不能重新获得天赋**，每次提供 3 个天赋供选择。
- 天赋分为 **A 级**（高级）和 **B 级**（普通），自己选过的天赋不会再次出现。
- 天赋抽取规则：
    + 第1个天赋：3个B级天赋中选1个
    + 第2-3个天赋：1个A级 + 2个B级中选1个
    + 第4个天赋：2个A级 + 1个B级中选1个
    + 第5个及以后：3个A级天赋中选1个

#### A 级天赋：
- **完美块：** 你的312视为万能牌
- **绝地反击：** 受到致命伤害时，血量降为1，下回合作战时分数短暂提升15%
- **占得先机：** 选牌阶段优先选牌
- **钢铁之躯：** 你受到的伤害降低30%
- **以退为进：** 你的7均视为6，4均视为3
- **致命魔术：** 造成伤害时有15%概率造成额外100%伤害
- **三相之力：** 你的下三张非选牌阶段的牌依次获得左/中/右单线癞子
- **紧急救援：** 跳过下一轮对战，立刻启动选牌阶段，你最先选择
- **我全都要：** 下次选择天赋时获得全部
- **潘多拉魔盒：** 随机获得两个B级天赋
- **利滚利：** 你每有125分，每回合加1分
- **戴森球：** 若6条长度3的连线均完成，则分数+6%
- **0号位：** 弃牌时，获得弃牌三个方向数字之和的临时分（持续1回合，可叠加）
- **坦诚相见：** 对战时仅比较双方盘面的基础连线分，无视双方全部额外分数加成
- **星河流转：** 位于2、4、7、13、16、18号位置的砖块获得长度3连线方向的单线癞子
- **冥想：** 每回合获得5点生命，直到单次连线获得25分及以上的分数
- **光波干涉：** 如果2条连线上的数字相同且连线长度也相同，这些连线获得20%额外分数
- **九转玄机：** 你的9视为癞子线
- **乾坤大挪移：** 你可以立刻交换盘面上的两个砖块
- **关键选择：** 从未获得的B级天赋中任选一个获得
- **生命游戏：** 在本局游戏内，每累计被扣20生命，额外+2分；每累计回复5生命，额外+1分
- **Y区域：** 1-5-10(左上)、10-14-17(右上)、10-11-12(垂直)每有一条线连接成功，你的总分增加2%

#### B 级天赋：
- **嗜血：** 战斗获胜时，生命值+4
- **还是有用的：** 你的每一行1分数额外加3，每一行2分数额外加6
- **快攻：** 战斗获胜时，对对手造成伤害+6
- **特立独行：** 分数为奇数时，战斗时分数短暂提升6
- **成双成对：** 分数为偶数时，战斗时分数短暂提升6
- **垃圾回收：** 每次弃牌时获得2分
- **来点实在的：** 立刻获得4分
- **攻击形态：** 造成的伤害增加15%，受到的伤害增加5%
- **防御形态：** 受到的伤害减少15%，造成的伤害减少5%
- **局部强化：** 10号位每个方向的数字完成一条匹配连线+3分（每方向仅一次，癞子可匹配任意数字，上限9分）
- **有舍有得：** 每战败4次，随机获得3枚砖块，从中选择1枚放置
- **三年之期：** 把接下来三个回合获得的砖块存起来，第四个回合时一次性摆放，并恢复存储期间受到的所有伤害
- **锻造：** 弃牌时，按顺序获得砖块三个方向数字的碎片，集齐三个方向的碎片合成一枚砖块放置
- **尾货处理：** 选牌时，如果你最后一个选，则可以获得剩下两个砖块
- **图灵测试：** 对战镜像战败也不会受到伤害
- **事不过三：** 你第4次战败时，免疫此次伤害
- **摇奖机：** 随机获得一个A级天赋
- **两极反转：** 你的1视为9，你的9视为1
- **败者之刃：** 你战败后获得4分临时分（可累积），在战胜一次后清除
- **临时用品：** 获得一个仅三回合可用的癞子
- **包扎：** 立即获得20点生命
- **百味草：** 获得1点中毒，你每因中毒损失1点生命，加1分
- **天使轮：** 获得15点临时分，直到下次完成一次连线为止
- **劫掠：** 击败玩家后，从他的盘面上随机挑选5个砖块，你选择其中一枚放置
- **多维抉择：** 天赋选择时，候选中额外提供1个A级与1个B级天赋
- **张三来袭：** 你每放置一张3，获得3点生命
- **贪婪宝藏：** 获得一张无法被覆盖的000，战败3次后变形为癞子
- **0的力量：** 获得一张000，只要000在场上存在，你放置卡牌时3视为4，6视为7
- **虚空之心：** 若你的10号位空置，则为你提供10点临时分
- **表演型人格：** 此后，你每在常规回合得分时，总分数-0.5%；每在选牌回合或额外卡牌放置得分时，总分数+3%
- **二环里：** 与10号位相邻的六个块若连接成功，这六个块朝向中心的方向各自变为单线癞子
- **恭喜栗子：** 你的中线连成4、9、7时，分别加4分、9分、7分
- **勃勃生机：** 立即获得15点生命。你的回血效果翻倍
- **一方通行：** 从X00、0X0、00X中选择一张牌
- **零风险投资：** 你的分数不会降低
- **以战代练：** 在你和分数差距5以内的对手对战后，你的分数永久+3
- **致命节奏：** 若没有弃牌，则获得1分；若弃牌，则清除通过本天赋获得的分数
- **时间锚：** 记录你当前的血量，在下次天赋选择时将血量调整为此数值
- **律动残余：** 你每次受到的伤害不超过25

### 游戏阶段
- 每个回合按以下顺序依次执行各阶段：
    1. **砖块放置阶段**：每位玩家获得一张砖块（普通轮从卡池1抽取，选牌轮从卡池2选取），选择放置位置或弃牌
    2. **战前额外砖块阶段**（如有）：处理战前触发的额外砖块，如「锻造」合成的砖块等
    3. **天赋选择阶段**（如有）：达成连线条件（每3条线）的玩家从3个天赋中选择1个
    4. **主动天赋阶段**（如有）：处理主动类天赋行动，也可 `pass` 放弃发动
    5. **对战阶段**（第3回合起）：随机配对进行分数比拼，分低者扣血，同时结算中毒伤害与淘汰判定
    6. **战后额外砖块阶段**（如有）：处理战后触发的额外砖块，如「三年之期」存储到期的砖块、「劫掠」从被淘汰玩家处获得的砖块等
- 所有阶段结束后进入下一回合，重复以上流程直到仅剩1名玩家存活


### 特殊事件
- 此外，每局游戏还可能随机出现特殊事件：
    + **大的要来了：**  卡池中没有1
    + **两极分化：**  卡池中没有5
    + **大的没了：**  卡池中没有9
    + **天降恩泽：**  第一轮每人发一个癞子
    + **调色盘：**  卡池中添加大量癞子
    + **有1吗：**  每行1额外加12分
    + **小透不算挂：**  提前公布下一轮的卡
    + **？？？：**  天降恩泽 + 调色盘 + 有1吗 + 小透不算挂
