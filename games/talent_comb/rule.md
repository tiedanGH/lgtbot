## 天赋云巢

- **游戏人数：**  2~8
- **原作：**  oh collei

### 游戏简介
- 本游戏基于云顶之巢改编，游戏的目标是通过在盘面中摆放卡牌构建连线，并在对战中击败其他玩家，成为最后存活者。
- 玩家需要在有限棋盘中规划布局，同时利用天赋与随机事件形成优势。

### 游戏流程
- 基础连线得分采用数字蜂巢规则：玩家每回合需要把一块数字砖块填入某个方格中（砖块允许被覆盖），当某条竖线或者斜线均为一种数字即可得分，得分数量为该数字乘以直线长度。
- **如果一行内仅包含癞子线，得分为长度×10。0数字线可以参与连线并计入连线数，但该连线得分为0。**
- 每名玩家初始血量为 **150**。从第3回合起，正常轮时结束随机挑选一名对手（奇数名玩家时会有一位玩家对战镜像），比拼当前分数，较低者会减去分数之差的血量（镜像的血量不会影响玩家本体）。
- 第8、15、22等轮（回合数+7）会从"卡池2"选出等于(人数+1)枚砖块并且公示（卡池2不包含癞子，若卡池2已空，则重置卡池重新抽取），由血量低往高进行选择，如果血量相同，则分数更低的玩家先进行选择。如果分数相同，由系统随机选择顺序。
- 血量归零玩家淘汰，剩余 1 人时游戏结束；卡池耗尽时所有存活者并列获胜。
- **最终积分完全按淘汰名次结算（和总分无关）**：每比一名玩家晚淘汰 +100 分，每比一名玩家早淘汰 −100 分。例如：4 人局完整分胜负的常规结果为 −300 / −100 / +100 / +300。
    + 同回合同阶段（同为对战、或同为中毒）淘汰的玩家名次完全相同、分数相同。
    + 中毒结算晚于对战，因此中毒淘汰名次 **优于** 同回合的对战阶段淘汰。
    + 卡池耗尽时所有存活玩家并列第一。

### 天赋系统
- 每完成 **3条连线** 可获得一次天赋选择，**破坏后重连不能重新获得天赋**，每次提供 3 个天赋供选择。
- 天赋分为 **A 级**（高级）和 **B 级**（普通），自己选过的天赋不会再次出现。
- 天赋抽取规则：
    + 第1个天赋：3个B级天赋中选1个
    + 第2-3个天赋：1个A级 + 2个B级中选1个
    + 第4个天赋：2个A级 + 1个B级中选1个
    + 第5个及以后：3个A级天赋中选1个
